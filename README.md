üìñ Project Overview

Passport Size Photo Maker is a browser-based tool for creating custom passport-sized photos. Users can upload an image, resize and crop it to exact dimensions, preview the result in real-time, and download it ready for official use.

No backend is needed; everything runs entirely on the client side.
