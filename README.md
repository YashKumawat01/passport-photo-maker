Passport Size Photo Maker

Author: Yash Kumawat

Technologies Used: HTML, CSS, JavaScript
Project Type: Frontend Web Application
Status: Complete

Project Overview

Passport Size Photo Maker is a lightweight, browser-based tool that allows users to create passport-sized photos of custom dimensions. Upload an image, adjust it to fit within a defined frame, and download the final output ready for official use.

Built entirely with HTML, CSS, and JavaScript‚Äîno backend required.

Features

Upload any photo (JPEG/PNG).

Resize and crop images to custom dimensions (e.g., 1.3 x 1.7 inches).

Preview changes in real-time.

Download the photo directly.

Fully client-side: works in any modern browser.

How It Works

Upload Image: Select a photo from your device.

Adjust Photo: Drag, scale, and position the image to fit the passport frame.

Preview: See the final photo instantly.

Download: Export the photo as an image file ready for printing.



Technologies & Concepts Used

HTML5: Page structure and layout

CSS3: Styling and responsive interface

JavaScript:

Image uploading (FileReader)

Dynamic resizing & cropping

DOM manipulation

Canvas API: Rendering final passport-sized image

Usage

Clone the repository:

git clone https://github.com/YashKumawat01/passport-photo-maker.git


Open index.html in your browser.

Upload and adjust your photo.

Click Download to save your passport-sized photo.

Future Improvements

Add preset sizes for different countries.

Background removal or color adjustment.

Export as print-ready PDF or high-DPI image.
